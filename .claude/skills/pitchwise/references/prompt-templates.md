# Prompt 模板库

## 模板选择逻辑

```
if 赛事包含 "世界杯" or "World Cup":
    → 使用 锦标赛预测模板
    → 注入: 分组形势 + 进球者数据
elif 赛事包含 "欧洲杯" or "Euro":
    → 使用 锦标赛预测模板
    → 注入: 分组形势
elif 赛事 in ["英超","西甲","意甲","德甲","法甲"]:
    → 使用 联赛预测模板
    → 注入: 历史特征数据（data/features_*2024.csv）
else:
    → 使用 通用预测模板
```

## 数据注入规则

- **历史特征**: 如果 `data/features_{league}2024.csv` 存在，注入主客场胜率、进球均值、近期状态
- **积分形势**: 杯赛小组赛 → 注入当前排名、出线条件；淘汰赛 → 注明"单场淘汰"
- **进球者**: 如果有已完赛数据，注入双方射手信息
- **伤病**: 如 WebSearch 返回具体伤病名单，注入；否则不编造
- **标签**: 所有预测结论标注"推断"，所有可验证事实标注数据来源

## 场地效应分析

世界杯 2026 部分高海拔/极端场地：

| 城市 | 球场 | 海拔 | 影响 |
|------|------|------|------|
| Mexico City | Estadio Azteca | **2,240m** | 空气稀薄，球速快 10-15%，体能消耗大 |
| Guadalajara | Estadio Akron | 1,566m | 中等海拔效应 |
| Monterrey | Estadio BBVA | 537m | 轻微海拔 |
| Denver | Empower Field | 1,609m | 中等海拔效应 |
| Miami | Hard Rock Stadium | 2m | 高温高湿（>30°C, >70%湿度） |
| Dallas | AT&T Stadium | 131m | 高温（>35°C），室内可控 |
| Houston | NRG Stadium | 13m | 极端高温高湿，室内可控 |

分析要点：
- **高海拔**（>1500m）：球飞行更快更远，适合远射和定位球。不适应高原的队伍下半场体能崩溃风险高。南美球队（玻利维亚/厄瓜多尔/哥伦比亚）在此类场地有天然优势。
- **高温高湿**（>30°C + >70% 湿度）：比赛节奏变慢，进球数通常偏低（高温下高强度压迫不可持续）。中场休息后的前 15 分钟关键。
- **室内球场**（AT&T, NRG, Mercedes-Benz 等 11 个球场）：温度恒定 22°C，消除天气变量。对技术流球队更有利。
- **中立场地**（所有锦标赛比赛）：无传统主客场优势。但美洲球队（墨西哥/美国/加拿大）仍有半个主场优势（时差+球迷+气候适应）。

注入规则：
- 如果比赛在海拔 > 1000m 的场地进行，在 prompt 中加入海拔警告
- 如果比赛在高温高湿的户外场地，加入天气因素
- 如果是室内球场，注明"室内恒温，无天气影响"
