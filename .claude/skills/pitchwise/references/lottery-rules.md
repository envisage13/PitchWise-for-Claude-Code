# 中国体育彩票竞彩足球规则

## 基本参数

| 规则 | 说明 |
|------|------|
| 每注金额 | 2 元 |
| 默认预算 | 100 元（50 注） |
| 截止时间 | 比赛开赛前（以销售终端为准） |
| 派奖方式 | 固定奖金（非交易所浮动赔率） |
| 最低串关 | 2 场（部分玩法可单关） |

## 玩法类型

### 1. 胜平负 (SPF)
- 预测全场 90 分钟结果：主胜(3) / 平局(1) / 客胜(0)
- 不含加时赛和点球
- **最常用玩法，适合作为策略主投**

### 2. 让球胜平负 (RQSPF)
- 强队让球后的胜平负
- 让球数标注为 -1, -2, +1 等
- 适合强队碾压局（德国 vs 库拉索这类）

### 3. 比分 (CRS)
- 精确预测最终比分
- 赔率高（通常 5.0-50.0+）
- 命中率低，适合小注博高赔
- **作为主投的补充/对冲**

### 4. 总进球数 (TTG)
- 预测总进球区间：0, 1, 2, 3, 4, 5, 6, 7+
- 适合做大小球判断时使用

### 5. 半全场 (HHFT)
- 预测半场和全场结果组合
- 9 种组合：胜胜/胜平/胜负/平胜/平平/平负/负胜/负平/负负
- 赔率较高，范围 3.0-30.0+

## 投注限制

| 限制 | 说明 |
|------|------|
| 单注最低 | 2 元 |
| 单票最高 | 视销售终端而定 |
| 单关最低 | 部分比赛可单关投注 |
| 混合过关 | 支持不同玩法串关 |
| 容错 | 支持 M串N 容错投注 |

## 策略约束

- 预算固定为 100 元 / 50 注（除非用户指定）
- 主投占比 60-70%，高赔博冷占比 10-20%，对冲占比 10-20%
- 单场最大投注不超过总资金的 25%（Kelly 约束）
- 连续亏损 3 次后自动降为 1/8 Kelly
